# FINEX Tax Preparation — Vercel Deployment (Final with varied reviews)

Includes Client Portal (TaxDome), Book Now (Calendly), Tax Academy, News & Updates, and updated Testimonials with three different reviews.

## Local Dev
```bash
npm i
npm run dev
# http://localhost:3000
```

## Deploy on Vercel
1. Push this folder to a GitHub repo.
2. Vercel → **New Project** → **Import Git Repository** → Framework: **Next.js** → **Deploy**.
